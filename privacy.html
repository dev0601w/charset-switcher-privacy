Charset Switcher Privacy Policy

Last Updated: 2026-04-26

Overview

Charset Switcher (hereinafter "this extension") is a Chrome browser extension for switching web page character encoding. This privacy policy explains how this extension handles user data.

Data Collection

This extension does not collect any user data.

  • No personal identification information collected
  • No browsing history collected
  • No web page content collected
  • No cookies or other tracking information collected
  • No data shared with any third parties

Local Data Storage

This extension uses Chrome Storage API (chrome.storage) to save the following settings locally in the user's browser:

  • Language preference - User's selected interface language
  • Encoding list configuration - User's custom encoding options to display
  • Site encoding preferences - User's encoding preferences for specific sites (optional feature)
  • Recently used encodings - Quick access to recently used encodings

All the above data is stored locally in the user's browser and is not uploaded or transmitted to any server.

Data Sharing

This extension does not share any data with any third parties. All data is stored only in the user's local browser.

Permissions Explained

This extension requests the following permissions and their purposes:

  • contextMenus - Display encoding switching options in the context menu
  • storage - Save user settings locally
  • activeTab - Get current tab information to switch encoding
  • scripting - Inject scripts into pages to change encoding
  • tabs - Access tab information
  • notifications - Display operation notifications

Third-Party Services

This extension does not use any third-party analytics, advertising, or tracking services.

Changes to This Policy

If there are any changes to this privacy policy, we will update the "Last Updated" date on this page.

Contact Us

If you have any questions about this privacy policy, please contact us through the Chrome Web Store extension page.

Charset Switcher 隐私政策

最后更新日期: 2026-04-26

概述

Charset Switcher（以下简称"本扩展"）是一款 Chrome 浏览器扩展，用于切换网页字符编码。本隐私政策说明本扩展如何处理用户数据。

数据收集

本扩展不收集任何用户数据。

  • 不收集个人身份信息
  • 不收集浏览历史
  • 不收集网页内容
  • 不收集 Cookie 或其他跟踪信息
  • 不与任何第三方共享数据

本地数据存储

本扩展使用 Chrome 存储 API（chrome.storage）在用户本地浏览器中保存以下设置：

  • 语言偏好设置 - 用户选择的界面语言
  • 编码列表配置 - 用户自定义显示的编码选项
  • 网站编码偏好 - 用户为特定网站设置的编码偏好（可选功能）
  • 最近使用编码 - 快速访问最近使用的编码

以上所有数据均存储在用户本地浏览器中，不会上传或传输到任何服务器。

数据共享

本扩展不与任何第三方共享任何数据。所有数据仅存储在用户本地浏览器中。

权限说明

本扩展请求以下权限及其用途：

  • contextMenus - 在右键菜单中显示编码切换选项
  • storage - 在本地保存用户设置
  • activeTab - 获取当前标签页信息以切换编码
  • scripting - 在页面中注入脚本以更改编码
  • tabs - 访问标签页信息
  • notifications - 显示操作通知

第三方服务

本扩展不使用任何第三方分析、广告或跟踪服务。

变更通知

如本隐私政策有任何变更，我们将在此页面上更新"最后更新日期"。

联系我们

如果您对本隐私政策有任何疑问，请通过 Chrome Web Store 扩展页面联系我们。
